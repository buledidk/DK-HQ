# sector-manufacturing-industry

Sector expertise for manufacturing, industrial, and general+services sectors.

## Sector-specific accounting

### Revenue (IFRS 15) — common patterns
- **Point-in-time**: manufacturing / retail / FMCG — deliver + control
- **Over-time**: prof services, construction, long-term contracts — input (cost-to-cost) or output method
- **Variable consideration**: rebates, volume discounts, returns, performance bonuses — expected value / most-likely; constraint
- **Principal vs agent** — marketplaces, franchising, distributors
- **Contract modifications** — separate vs catch-up vs termination + new
- **Warranties** — assurance vs service split
- **Licences** — right-to-use vs right-to-access
- **SaaS customisation** — not distinct unless another vendor could perform without significant modification

### Inventory (IAS 2)
- Cost formula (FIFO / WAC) — consistent per class
- Standard cost OK if approximates actual; variances allocated (inventory vs COGS)
- NRV per item/category; ageing + obsolescence provisions
- Borrowing costs not capitalised (except IAS 23 qualifying assets — long-duration)
- Retail inventory method permitted if approximates cost

### PP&E (IAS 16)
- Componentisation — separate material components (aircraft engines, ship hulls, building services)
- Capitalisation thresholds
- Major overhauls capitalised; repairs expensed
- Impairment triggers (IAS 36)
- Grants (IAS 20) — gross vs net

### Intangibles + R&D (IAS 38)
- Research always expensed
- Development capitalised only if 6 criteria met
- Internally-generated brands / customer lists prohibited
- Software dev — capitalise in dev phase when criteria met
- Amortisation straight-line; impairment tests

### Lease accounting (IFRS 16 / ASC 842)
- Lease identification ¶9
- Lessee: RoU + liability at PV (IBR)
- Term: non-cancellable + options reasonably certain
- Modifications: separate contract vs re-measurement
- Lessor: operating vs finance
- Sale-and-leaseback: IFRS 15 PO first, then lease

### Provisions (IAS 37)
- Warranty: present obligation at sale; prob-weighted expected outflow
- Restructuring: detailed formal plan + valid expectation raised
- Onerous contracts: unavoidable costs > economic benefits
- Decommissioning: PV of estimated cost; re-measure for estimate + discount-rate changes

### Deferred tax (IAS 12)
- DTL on taxable temp diffs (exceptions for goodwill, initial recognition)
- DTA only if probable taxable profit
- Pillar 2 top-up: disclose current-period impact

### Impairment (IAS 36)
- Indicators: mkt cap < BV, loss forecasts, strategic/tech change
- CGU at lowest level of independent cash inflows
- Recoverable = max(FVLCD, VIU)
- VIU DCF pre-tax CF + pre-tax rate, max 5-yr explicit + terminal
- Goodwill impairment never reverses; asset can reverse

### Long-term contracts (IFRS 15 over-time)
- Cost-to-cost % completion
- Contract asset (positive %C - billings) / liability (negative)
- Loss-making: recognise expected loss immediately
- Variations + claims: constrain

### Rate-regulated (IFRS 14)
Interim standard for first-time adopters; most follow local GAAP pending IASB.

## Sector audit focus

### Inventory
- Year-end physical count observation (ISA 501)
- Cut-off (last GRN + last despatch)
- Cost — recompute per policy + overhead absorption
- NRV — subsequent selling prices, ageing, specific obsolescence
- Consignment / bill-and-hold — existence + ownership

### Revenue
- Pattern per sector (IFRS 15 5-step)
- Cut-off — invoice / GRN / despatch alignment
- Rebates + returns + warranties vs history
- Channel stuffing — quarterly spikes + reversal
- SSP methodology + inputs

### PP&E + intangibles
- Additions / disposals — approval + evidence + threshold
- Impairment trigger scan, CGU-level DCF
- Useful lives vs peers + technology change
- R&D capitalisation criteria on sample

### Long-term contracts
- Cost-to-complete: sample engineers' estimates, back-test PY
- Variations + claims: recognise only if highly probable
- Contract loss provisions
- Project-health dashboard: budget vs actual + forecast

### Decommissioning
- Methodology + peer benchmarking
- Discount rate — risk-free + premium
- Useful-life changes → re-measurement
- Utilisation vs plan

### Going concern — sector stressors
- Consumer demand shock (retail/FMCG)
- Commodity shock (O&G, metals)
- Supplier failure (supply-chain concentration)
- Capex + refinance risk
- Covenant definitions sector-specific

## Operational KPIs

### Manufacturing
OEE, first-pass yield, scrap rate, inventory turns, DIO, DPO, DSO, CCC,
gross margin by product line.

### Retail
LFL sales, basket size + conversion, gross margin (stock-adjusted),
shrinkage, store-level EBIT.

### SaaS / Tech
ARR/MRR, NRR/GRR, CAC + LTV, Rule of 40, gross margin %, ARPU/ACV.

### Construction
Backlog / book-to-bill, % complete vs billed, margin at completion, cost
overrun %.

### Oil & gas
Reserves (1P/2P/3P), F&D cost, opex per boe, netback, decline rate.

### Pharma
Pipeline by phase, peak sales potential, R&D intensity, gross-to-net,
days on hand.

## Automation opportunities

1. Inventory — cycle counts + ERP reconciliation + aged/obsolete
   detection + auto-provision
2. Three-way match (P2P) — PO/GRN/invoice tolerance + exception flow
3. Revenue engine (IFRS 15) — 5-step per-contract + SSP + over-time % + VC constraint
4. Rebate + returns accruals — expected-value model + back-test
5. SaaS metering + billing — usage → invoice → revenue schedule
6. Capital project tracking — EVM (scope + schedule + cost)
7. Impairment monitoring — trigger scan + CGU DCF + sensitivity
8. Lease register + IFRS 16 calc — contract ingestion + RoU + liability + modifications
9. Warranty engine — per-product expected cost + utilisation
10. Decommissioning schedule — PV + rate sensitivity + re-measurement triggers
11. Supplier-claim + contract-variation tracker (construction + infra)

## Anti-patterns

- Standard-cost variances ignored as "immaterial" without quantification
- Manual JE-driven inventory provisions without a model
- Revenue at invoice regardless of control transfer
- Dev-cost capitalisation without documented IFRS 38 criteria test
- Channel-stuffing dismissed as "normal quarter-end"
- Impairment indicator scan only annually (should be continuous)
- Lease modifications as new contracts without IFRS 16 ¶44–46
- Decommissioning estimates not re-measured for rate changes
- Gift-card / loyalty breakage on history without IFRS 15 expected-value + constraint
