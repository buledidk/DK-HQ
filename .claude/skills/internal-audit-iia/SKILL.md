# Internal Audit (IIA GIAS)

Built for Chief Audit Executives and IA managers operating under the 2024 IIA Global Internal Audit Standards.

## When to use

- Building / refreshing the IA charter, strategy, or annual plan.
- Running a risk-based IA engagement (planning → fieldwork → report).
- Designing the quality assurance & improvement program (QAIP).
- Preparing for an external quality assessment (EQA).
- Reporting to Audit Committee / Board / Risk Committee.
- Building combined assurance map across 1st/2nd/3rd lines.

## Standards in scope

- IIA Global Internal Audit Standards (GIAS) 2024 (effective Jan 2025), replacing IPPF.
- IIA Three Lines Model 2020.
- UK Financial Services IA Code (Chartered IIA) — mandatory for FS firms.
- CIIA Risk in Focus (annual hot topics).
- IFACI / ECIIA position papers.

## IA engagement workflow

| Phase | Key tasks | Output |
|---|---|---|
| 1 · Planning | Scope, objectives, criteria, risk register cross-walk, stakeholder briefing | Engagement planning memo, fieldwork programme |
| 2 · Fieldwork | Walkthroughs, control testing, substantive analytics, interview logs | Evidence file, finding registrar |
| 3 · Issue agreement | Rate findings (High/Med/Low), agree root cause + management action owner + date | Findings summary |
| 4 · Reporting | Exec summary (one-pager), detailed findings, opinion grade, management response | IA report, board summary |
| 5 · Follow-up | Track actions to closure, evidence-back validation, overdue escalation | Open-actions dashboard |

## Audit opinion grades (typical)

- **Satisfactory** — controls designed and operating effectively.
- **Limited Assurance** — control weaknesses with material impact.
- **Unsatisfactory** — pervasive failures.
- **No opinion** — scope-limited review.

## Annual planning

1. **Audit universe** — every entity × process × geography × IT system.
2. **Risk assessment** — inherent × control × velocity × regulatory change; cross-walk to ERM risk register.
3. **Heatmap** — plot universe on risk score × assurance coverage.
4. **Cyclical coverage** — mandatory (SOX, Consumer Duty, SMCR certification), high-risk, rotational.
5. **Resource plan** — FTE × skill matrix; co-source needs (cyber, model audit, data science).
6. **Plan approval** — AC chair sign-off, CAE 1:1 with chair quarterly.

## Three Lines Model operationalisation

| Line | Owner | Role |
|---|---|---|
| 1st | Business management | Own risks, design + operate controls |
| 2nd | Risk, compliance, InfoSec, finance control | Oversee, challenge, set framework |
| 3rd | Internal audit | Independent assurance on 1st + 2nd |
| Governance | Board + AC | Hold all lines accountable |

Combined assurance map = rows (risks) × columns (lines) with coverage heat.

## QAIP (mandatory under GIAS 2024)

- **Internal**: ongoing engagement-level QA review + periodic self-assessment.
- **External**: EQA every 5 years, independent qualified assessor, public statement of conformance.
- Metrics: on-time plan execution %, finding acceptance rate, action closure on time %, stakeholder satisfaction NPS, CPE hours per auditor.

## UK FS IA Code — additional requirements

- Root-cause analysis mandatory.
- Review of risk + control culture.
- Report on whether second line is effective.
- Specific board + regulator reporting expectations (PRA thematic reviews).

## Deliverables

- IA charter (board-approved).
- Annual risk-based audit plan + quarterly re-forecast.
- Engagement planning memo template.
- Findings rating matrix + root-cause taxonomy.
- Board / AC reporting pack (executive summary + thematic view).
- QAIP framework + EQA readiness file.
- Combined assurance map.

## Anti-patterns

- Audit plan driven by "last year's plan + 5% delta" instead of risk.
- Findings without root cause → repeat findings.
- No follow-up discipline → action register decays.
- CAE not independent (reports to CFO instead of AC chair).
- QAIP treated as annual tick-box.

## Standards anchor

IIA GIAS 2024, IIA Three Lines Model 2020, UK Financial Services Internal Audit Code (Chartered IIA), ISO 31000 (risk mgmt), COSO ERM 2017, UK Corporate Governance Code 2024.
