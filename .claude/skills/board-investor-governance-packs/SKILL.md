# board-investor-governance-packs

Draft board / investor / regulator packs that land clearly, survive
challenge, and stand up to audit review. Aligned to UK Corporate
Governance Code 2024, FRC Guidance, FCA / PRA expectations, and IR best
practice.

## Pack taxonomy + audience fit

| Pack | Audience | Frequency | Length | Voice |
|------|----------|-----------|--------|-------|
| **Board pack** | NEDs + ExCo | Monthly / bi-monthly | 80–150 pp | Strategic, balanced, forward |
| **Audit Committee** | AC members (NEDs) | Quarterly | 40–80 pp | Risk / controls / FS / audit |
| **Risk Committee** | RC + CRO | Quarterly | 30–60 pp | Top risks, KRIs, RAS |
| **ALCO** (banks) | Treasury + CFO + Risk | Monthly | 20–40 pp | Capital, liquidity, funding, IRRBB |
| **RemCo** | NEDs + CPO + Chair | Semi-annual | 20–40 pp | Pay, performance, MRTs |
| **NomCo** | NEDs + CEO + Chair | As-needed | 10–30 pp | Composition, succession, D&I |
| **Earnings pack** | Investors + analysts | Quarterly / semi-annual | 30–50 pp | Results, outlook, drivers |
| **Analyst briefing** | Sell-side + buy-side | Semi-annual + ad-hoc | 20–40 pp | KPIs, guidance, sensitivities |
| **AGM pack** | Shareholders | Annual | 30–60 pp | Resolutions, remuneration, auditor |
| **Regulator pack** | PRA / FCA / FRC / HMRC | As requested | Variable | Fact-based, reserve nothing, owner+date |

## Universal structure ("the BIG-4 pattern")

Every good pack answers four questions in this order:

1. **B**ottom line — what happened, what we recommend, what we need
2. **I**mplications — financial, strategic, regulatory, people, reputational
3. **G**round truth — evidence + data + methodology
4. **4**th ask — decision / approval / noting / information

Front-load the ask. Assume readers have 7 minutes per paper.

## Board pack anatomy (UK Code + FRC Guidance)

### Standing items every board
1. Chair's opening + prior minutes
2. CEO report — strategy, market, material events, people
3. CFO report — FP&A vs budget + PY + consensus, cash, debt, covenants, capex, WC, outlook, KPI dashboard
4. Principal risks — RAS position, KRI breaches, emerging
5. Committee updates — AC, RC, RemCo, NomCo decisions
6. Regulatory + legal + s172 considerations
7. People + culture — attrition, D&I, succession, conduct
8. Operational resilience — IBS status, incidents
9. Stakeholder map — s172 anchor

### Decision paper template
- Cover sheet (number, date, author, sponsor, classification, recommendation, risk rating)
- Executive summary ≤ 1 page (BIG-4 answers)
- Context + purpose ≤ 2 pages
- Options table (strategy fit, financials, risk, time, regulatory, reputation)
- Recommendation + rationale
- Risks + mitigations
- Implementation plan (dates, owners, KPIs)
- Financial impact (P&L, cash, capital, IRR, payback)
- Stakeholder considerations (s172 ledger)
- Appendices

## Audit Committee pack — standing agenda

### Every meeting
- Previous minutes + action tracker
- External audit (plan/progress/findings/mgmt letter/fees/independence)
- Internal audit progress + findings + remediation
- Whistleblowing (routes, volumes, themes, closures)
- Financial-reporting oversight (judgments, estimates, policy changes, GC view)
- Risk + control environment (from RC + CFO)
- Regulatory correspondence (FRC, PRA/FCA, HMRC)

### Year-end specific
- Going-concern + stress scenarios
- Viability (UK Code Provision 31)
- Fraud risk reassessment + JE testing summary
- Auditor's report walkthrough (opinion, KAMs, EoMs, modifications)
- Annual-report fair-balanced-understandable
- FRC audit-firm governance code compliance

### ISQM-visible
- EQR confirmation, archive status (60-day), RCA of prior findings

## Risk Committee pack

1. RAS — position vs limits + triggers + breaches
2. Principal risk profile — heatmap + trend per risk
3. KRI dashboard — RAG + breach + actions
4. Emerging risks — horizon scan + owners
5. Control effectiveness — heatmap + deficiency severity
6. Incidents + near-misses — pattern analysis + RCA
7. Third-party risk — SOC 2 + continuous monitoring
8. Cyber + op resilience — threats, IBS tolerance, test results
9. Model risk — inventory + tier + validation + findings
10. Regulatory change — incoming rules + gap + plan

## ALCO pack (banks + building societies)

1. Capital — CET1, T1, Total, leverage, Pillar 2 headroom, forecast
2. Liquidity — LCR, NSFR, HQLA composition, concentration, maturity ladder
3. Funding — deposit composition, retail vs wholesale, cost + rate sensitivity, run-off back-test
4. IRRBB — ∆EVE, ∆NII, behavioural assumptions
5. Treasury P&L + FX exposure
6. Stress testing — latest vs thresholds
7. Recovery + resolution — RPI, MREL, TLAC
8. Forecast — base + stress 1–3 years
9. Climate — transition + physical, NZBA

## Investor pack (earnings / analyst)

### Earnings deck anatomy
1. Headline — 6–8 KPIs vs guidance + PY
2. Strategic progress — 3 things we said, 3 things we did
3. Financial performance — segment, region, drivers (bridge)
4. Balance sheet + cash flow — leverage, FCF, capital allocation
5. Operational KPIs — sector-specific
6. Outlook + guidance — ranges + assumptions + risk factors
7. Appendix — reconciliations, definitions, history

### Non-negotiables
- APM reconciled to GAAP (ESMA)
- Consistency with filed accounts
- Forward-looking safe-harbour
- MAR — no selective disclosure; insider procedures
- Promises tracked (Consumer Duty outcomes if FS)

## Regulator pack (s166, Dear CEO, MRA)

- Executive summary — what we heard, what we're doing, by when, by whom
- Root-cause analysis (not symptoms)
- Action plan (SMART, owner, date, evidence)
- Governance of remediation (exec sponsor, steering, reporting)
- Independent validation (IA or external)
- Status dashboard — RAG per action, evidence link
- Lessons learnt + systemic fix
- Risk of recurrence + preventive controls

## Drafting discipline

### Voice
- Third-person neutral for board/AC/regulator packs
- First-person plural for investor packs
- No hedging that hides truth; no unexplained jargon
- Active voice; short sentences

### Numbers
- Every number traces to source
- APMs reconciled on same page or immediate appendix
- Sensitivities where assumptions drive conclusion
- Direction + magnitude + cause for each variance

### Evidence
- Every claim sourced (WP, system, regulator paragraph)
- Forward-looking labelled with assumptions

### Pack engineering
- Cover + paper numbering + author + sponsor
- ToC + page numbers
- Templates version-controlled
- Dispatch ≥ 5 clear working days
- No track-changes in final pack
- Classify every page (Public/Internal/Confidential/Strictly Conf.)

## Anti-patterns

- Papers > 8 pages without TL;DR
- Decision papers without ask on page 1
- KPIs without target + PY comparison
- Narrative inconsistent with numbers
- APMs more prominent than GAAP
- Inside info in investor pack without MAR treatment
- "Noting" papers that should be decisions
- Remediation without independent validation
